Minesweeper Project

Minesweeper Python Project

This project is a recreation of minesweeper, starting from a blank slate and using python with tkinter for the user interface. This project let me practice python skills, implement a graphical user interface, and study run time complexity in terms of creating the minesweeper game board.

Click "Run" on the repl to play the game. It might not load immediately, which can sometimes be fixed by refreshing the page. Once the program starts, there is a drop down menu to change the size of the game. Once started, left click to clear tiles and right click to flag a tile. "Restart" resets the existing game and "New Game" returns to the main selection menu.
