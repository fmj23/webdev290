Minesweeper Python Project

I created this project during an algorithms class with the purpose of studying the time complexity of a program to verify correct solutions.

To play the game, click "Run" on the repl (might have to refresh the browser for the module to actually load) and choose the puzzle size from the drop down menu. Then press start. Left click to clear tiles, and right click to flag a tile. "Restart" resets the existing puzzle and "New Game" returns to the puzzle selection menu.
